# Энергодом — калькулятор расхода электроэнергии

Статический сайт для **ориентировочного** расчёта потребления электроэнергии в загородном доме: дневной и ночной тариф, режим проживания, сезонный коэффициент, разбивка по помещениям, пиковая нагрузка, график и экспорт отчёта.

## Структура репозитория

| Файл | Назначение |
|------|------------|
| `index.html` | Приветственный лендинг и переход к калькулятору |
| `calculator.html` | Полный функционал: таблица приборов, итоги, печать, скачивание `.txt` |
| `Текстовый документ.html` | Исходная рабочая версия (можно удалить после переезда на `calculator.html`) |

## Как открыть локально

1. Склонируйте или скачайте папку проекта.
2. Откройте в браузере **`index.html`** (двойной щелчок или «Открыть с помощью…»).

Для корректной работы всех путей откройте файлы через локальный сервер (опционально):

```bash
# из корня проекта, если установлен Python 3
python -m http.server 8080
```

Затем перейдите в браузере на `http://localhost:8080/`.

## Развёртывание

Проект не требует сборки: загрузите содержимое папки на **GitHub Pages**, **Netlify**, **Cloudflare Pages** или любой статический хостинг. Главная страница входа — `index.html`.

## Тема оформления

Переключатель **светлая / тёмная** тема сохраняется в `localStorage` под ключом `eco-calc-theme` и синхронизируется между `index.html` и `calculator.html`.

## Ограничения

Расчёт носит рекомендательный характер. Реальное потребление зависит от климата, утепления, режимов техники и качества данных о часах работы приборов.
